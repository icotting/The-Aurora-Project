Richmond Daily Dispatch Places

Description

This data set represents a relational model extracted from the Richmond Daily Dispatch (see citation) XML files and reviewed as part of the Aurora Project. The data set includes place names (city, state, county, country etc.) extracted from volumes of the Richmond Daily Dispatch between October 31, 1860 and Febuary 9, 1866. The actual text associated with places is also included for reference and context analysis.

Schema

The data set consists of the following 6 tables:

alt_places

Field Type Description
PlaceOccurrence_ID bigint the ID of the occurrence of a named place
alternatePlaces_ID bigint The ID of a possible alternate place in the master place name table

dispatchvolume

Field Type Description
ID bigint Identifier for a published volume of the newspaper
filename varchar The name of the XML file from the original dataset
volumedate date The date on which the volume was published

place

Field Type Description
ID bigint Identifier for the unique place
reviewed tinyint A boolean flag indicating whether or not the place reference was reviewed
legacyid varchar The unique identifier of the place in the original XML dataset
name varchar The place name
longitude float The x coordinate of the place
latitude float The y coordinate of the place
deleted tinyint A boolean flag indicating whether or not the reviewer wanted the place removed

placeoccurrence

Field Type Description
ID bigint Identifier for occurrence of a place in text
reviewed tinyint A boolean flag indicating whether or not the place occurrence was reviewed
actualtext varchar The text text for the place name as found in the text of the newspaper
sentence_id bigint The foreign key to the sentence table linking to the sentence in which the place occurrence is found
place_id bigint The foreign key to the place table for the specific occurrence

sentence

Field Type Description
ID bigint Identifier for the sentence extracted from a dispatch volume
text varchar The text of the sentence
number int The sentence number of the dispatch volume
volume_id bigint The foreign key dispatchvolume table for the volume in which the sentence is found

reviewnote

Field Type Description
ID bigint Identifier for the note provided by a reviewer for a place or place occurrence
action varchar A description of the action taken by the reviewer (No Action, Delete Point, Needs Additional Review)
reviewedon date The date on which the place or place occurrence was reviewed
comment varchar Any comments provided by the reviewer to support their action
place_id bigint The foreign key to the place table
occurrence_id bigint The foreign key to the place occurrence table
