Freedmen's Bureau Labor Statistics

Description

The Freedmen's Bureau Labor Statistics data set contains detailed records for work contracts issued to emancipated slaves between Febuary 24, 1865 and July 14, 1867. Each record provides details about the work being contracted, the individual to whom the contract was given, the hiring office issuing the contract, and the location where the work was to be performed.

Schema

The data set consists of the following 4 tables:

contract

Field Type Description
ID bigint Identifier for the contract entry
COMMENTS varchar Misc. text comments for the contract
CONTRACTDATE date The date the contract was issued
EMPLOYERAGENT varchar The name of the employer
LENGTHOFSERVICEMONTHS varchar The duration of the contract - this field is a text field to accomodate issues from the original dataset
POSITION varchar The type of work being contracted
RATEOFPAY varchar The amount paid for service provided
RENUMERATION varchar Description of any additinal payment (room and board etc.) for service provided
WORKCLASS varchar The type of work / service provided
DESTINATION_ID bigint The foreign key to the destination table
OFFICE_ID bigint The foreign key to the hiring office table
WORKER_ID bigint The foreign key to the worker table

contractdestination

Field Type Description
ID bigint Identifier for the contract destination
COUNTY varchar The county in which the work or service were provided
STATE varchar The state in which the work or service were provided
TOWNSHIP varchar The township in which the work or service were provided
LATITUDE float The y coordinate of the location represented by the destination - a centroid value for state or county without township
LONGITUDE float The x coordinate of the location represented by the destination - a centroid value for state or county without township

hiringoffice

Field Type Description
ID bigint Identifier for the hiring office
NAME varchar The name of the office
LATITUDE float The y coordinate of the hiring office location
LONGITUDE float The x coordinate of the hiring office location

worker

Field Type Description
ID bigint Identifier for the worker
FIRSTNAME varchar The first name of the worker
LASTNAME varchar The last name of the worker
GENDER varchar The gender of the worker
LABEL varchar A misc. lable field generally containing the first and last name of the worker
